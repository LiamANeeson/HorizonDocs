 CHzn::BufferLayoutDefine the layout of a Vertex Buffer using the Buffer Layout. Buffer Layouts are required because Vertex Buffers are essentially an array of primitive data (usually Float). We need to tell the GPU, how should it interpret this data. Hence, the Buffer Layouts define to the GPU how to interpret the data held by the Vertex Buffer. A Buffer Layout is represented by the list of Buffer Elements. It is important to note that the Layout is interpreted in the order by which the user has added Buffer Elements to the Layout.
