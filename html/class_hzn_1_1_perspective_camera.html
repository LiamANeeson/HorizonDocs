Detailed Description

Represents a Perspective Camera (Usually used as a 3D camera).

Create a Perspective Camera.

Parameters
aspectRatio: Provide the aspect ratio of the screen, which can be calculated as width / height of the screen.
fov: Default value is 45 degrees. Field of View of the Camera (in degrees), increasing the field of view zooms out and decreasing zooms in. Usually ranges from 45 degrees to 65 degrees
nearPlane: Default value is 0.1f, The near plane of the perspective projection, a 2D plane on which objects in the 3D world are projected.
farPlane: Default value is 100.0f, The far plane of the perspective projection, anything object this plane is not visible in the given frame.
position: Defaults value is (0, 0, 0). Position of the Camera in the world.
