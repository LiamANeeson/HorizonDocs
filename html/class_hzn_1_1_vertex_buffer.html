Detailed Description

Vertex Buffer is essentially a list of vertices. Each Vertex has Attributes such as position (where it would be drawn on the screen) and many more Attributes such as the color of the Vertex, Texture Coordinate that maps to the Vertex, etc. But the data held by a Vertex Buffer is essentially just a list of floats. We use a Buffer Layout to define how this array of floats should be interpreted by the Shaders and the Graphics Card to use and perform operations based on the data provided to the Vertex Buffer.
