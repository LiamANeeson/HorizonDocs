Detailed Description

Represents a Buffer Element. Holds information about the Attribute/Elements in a Vertex. Buffer Element is made up of the name that would be used to represent the Buffer Element of a Vertex in Shader, the 'ShaderDataType' used to represent the element, size of the 'ShaderDataType', offset of the Buffer Element, which is the starting position of the element in a Vertex and a bool value that indicates if the values of the element would be normalized.
