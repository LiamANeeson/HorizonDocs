◆ MonoGCRootSource

enum MonoGCRootSource

This enum is used by the profiler API when reporting root registration.

Enumerator
MONO_ROOT_SOURCE_EXTERNAL 

Roots external to Mono. Embedders may only use this value.

MONO_ROOT_SOURCE_STACK 

Thread call stack.

The key parameter is a thread ID as a uintptr_t.

MONO_ROOT_SOURCE_FINALIZER_QUEUE 

Roots in the finalizer queue. This is a pseudo-root.

MONO_ROOT_SOURCE_STATIC 

Managed static variables.

The key parameter is a MonoVTable pointer.

MONO_ROOT_SOURCE_THREAD_STATIC 

Managed static variables with ThreadStaticAttribute.

The key parameter is a thread ID as a uintptr_t.

MONO_ROOT_SOURCE_CONTEXT_STATIC 

Managed static variables with ContextStaticAttribute.

The key parameter is a MonoAppContext pointer.

MONO_ROOT_SOURCE_GC_HANDLE 

GCHandle structures.

MONO_ROOT_SOURCE_JIT 

Roots in the just-in-time compiler.

MONO_ROOT_SOURCE_THREADING 

Roots in the threading subsystem.

The key parameter, if not NULL, is a thread ID as a uintptr_t.

MONO_ROOT_SOURCE_DOMAIN 

Roots in application domains.

The key parameter, if not NULL, is a MonoDomain pointer.

MONO_ROOT_SOURCE_REFLECTION 

Roots in reflection code.

The key parameter, if not NULL, is a MonoVTable pointer.

MONO_ROOT_SOURCE_MARSHAL 

Roots from P/Invoke or other marshaling infrastructure.

MONO_ROOT_SOURCE_THREAD_POOL 

Roots in the thread pool data structures.

MONO_ROOT_SOURCE_DEBUGGER 

Roots in the debugger agent.

MONO_ROOT_SOURCE_HANDLE 

Roots in the runtime handle stack. This is a pseudo-root.

The key parameter is a thread ID as a uintptr_t.

MONO_ROOT_SOURCE_EPHEMERON 

Roots in the ephemeron arrays. This is a pseudo-root.

MONO_ROOT_SOURCE_TOGGLEREF 

Roots in the toggleref arrays. This is a pseudo-root.
